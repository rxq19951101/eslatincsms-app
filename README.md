# App - 充电桩用户端应用

Expo React Native 移动应用，用户端充电桩使用应用。

## 功能特性

- ✅ 扫码充电
- ✅ 地图查看充电桩位置
- ✅ 充电历史记录
- ✅ 消息支持
- ✅ 用户账户管理

## 快速开始

### 开发模式

```bash
cd app
npm install
npm start
```

### 构建生产版本

```bash
# Android
npx expo build:android

# iOS
npx expo build:ios
```

## 环境变量

```bash
EXPO_PUBLIC_CSMS_API_BASE=https://api.yourdomain.com
```

## 依赖

- Expo SDK
- React Native
- React Navigation
- Expo Camera
- React Native Maps
